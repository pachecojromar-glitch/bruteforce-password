# Brute Force Password - Educational Project

## Description
This project demonstrates a simple brute force password attack implemented in Python.

The objective is to understand how weak passwords can be easily broken and why strong password policies are important.

## How to run

1. Clone the repository:
   git clone https://github.com/pachecojromar-glitch/bruteforce-password.git

2. Enter the directory:
   cd bruteforce-password

3. Run the script:
   python bruteforce.py

## Example Output

Contraseña encontrada: abc  
Intentos realizados: 731  
Tiempo de ejecución: 0.0021 segundos  

## Reflection

When password length increases, the number of possible combinations grows exponentially.

Strong passwords (8+ characters with uppercase, lowercase, numbers and symbols) can take years to crack using brute force.
